# 🗂️ Drag-and-Drop Kanban Board

A fully functional Kanban board built with **vanilla HTML, CSS, and JavaScript** — no libraries or frameworks used.

##  Features Implemented

 1  HTML Drag & DropNative API only — cards snap to correct position between existing cards 
 
 2 Add Card Each column has an Add Card button with title, description, priority, and confirm/cancel 
 
 3 Inline Editing Double-click any card title or description to edit; Enter saves, Escape cancels |

 4 Priority Badges P1 = Red, P2 = Orange, P3 = Blue, P4 = Gray — dropdown on every card to change priority 
 
 5 localStorage Persistence Board state saves on every change and survives page refresh 
 
 6 earch & Filter Real-time search bar + P1/P2/P3/P4/All filter buttons; non-matching cards fade out 
 
 7 Keyboard AccessibilityTab to focus cards, Space/Enter opens context menu to move or delete without a mouse
